# 🌐 CLI Agents HQ - Frontend

This is the React-based management dashboard for CLI Agents HQ. It features an isometric 3D office interface for orchestrating multiple AI agents.

## 🚀 Installation & Setup

1. **Prerequisites:**
   - Node.js (v18+)
   - Access to the backend server (Plesk or Local)

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Development Mode:**
   Starts the Vite dev server with Hot Module Replacement (HMR).
   ```bash
   npm run dev
   ```

4. **Production Build:**
   Generates a highly optimized build in the `dist` folder.
   ```bash
   npm run build
   ```

## 🛠️ Technology Stack
- **Framework:** React 19 (TypeScript)
- **Bundler:** Vite
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Real-time:** Socket.io-client
